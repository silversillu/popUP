Easy to use and customize popUP

Getting started

To get started all we need is define a new popUP, ours is called "basic"

                basic = popUP('basic');

If we do not want to change any settings, all we need to do next is to call the popUP, in our case "basic"

                    basic.add();

To activate our new popUP, just add a class "popUP_activator" to any element you wish!

Activate "basic"

You can name your popUPs whatever you like. The variable is only used to create and modify the popUP. The argument that you give to the popUP() function will be the name of your popUP and is also used for activating and hiding the popUP!

To activate your popUP your trigger must have a class "name_activator" ("name" being the name you added into the popUP() function).

Content:

While defining a new popUP, you would probably want to add some content to your popUP.

Here we define a new popUP with content

                    example1 = popUP('example1');
                    example1.content = 
                        "
Hello I am a custom popUP window!
"+
                        "
You can add any HTML code in me!
";
                    example1.add();
Activate "example1"

Content could also be an element from page, for example the following paragraph that has a class "element-from-page":

"I will be in a popUP"

                    example2 = popUP('example2');
                    example2.content = $('.element-from-page');
                    example2.add();
Activate "example2"

Settings:

While defining a new popUP, there are a few things that you would like to change.

Option Default Description
content none HTML or jQuery selector for element from page.
activator name_activator Custom class for element which will trigger your popUP when clicked.
hider name_hider Custom class for element which will hide your popUP when clicked
hiderContent X Content for your default hider
customSyles none Add your custom CSS
customStylesFile none Link to your custom CSS file
overlay false Choose true to activate overlay for your popUP
theme default Choose from any default themes
clickAnywhereClose false When true, the popUP will be hidden when user clicks anywhere outside of the popUP.

Here is we define a new popUP with some custom settings

                    example3 = popUP('example3');
                    example3.content = 
                        "
Hello I am a styled popUP window!
"+
                        "
I"+
                        " am"+
                        " so"+
                        " cool
";
                    example3.customStyles = 
                        ".red {color: #DB4437}"+
                        ".yellow {color: #F4B400}"+
                        ".green {color: #0F9D58}"+
                        ".blue {color: #4285F4}"+
                        ".example3.popUP-container {background: #FDF0F0}";
                    example3.add();
Activate "example3"

Demos

Image with overlay

To create a beautiful popUP with image just add your < img> tag into the content. You can also add overlay and optional click anywhere to hide.

                    example4 = popUP('example4');
                    example4.content = "";
                    example4.overlay = true;
                    example4.clickAnywhereClose = true;
                    example4.add();
