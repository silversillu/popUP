Settings:

While defining a new popUP, there are a few things that you can change:

Option Default Description
content none HTML or jQuery selector for element from page.
activator name_activator Custom class for element which will trigger your popUP when clicked.
hider name_hider Custom class for element which will hide your popUP when clicked
hiderContent X Content for your default hider
customSyles none Add your custom CSS
customStylesFile none Link to your custom CSS file
overlay false Choose true to activate overlay for your popUP
theme default Choose from any default themes
clickAnywhereClose false When true, the popUP will be hidden when user clicks anywhere outside of the popUP.
